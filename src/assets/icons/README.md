# Icônes PWA

Ce dossier doit contenir les icônes PNG générées à partir de `icon-template.svg`.

## Fichiers requis

- `icon-72x72.png`
- `icon-96x96.png`
- `icon-128x128.png`
- `icon-144x144.png`
- `icon-152x152.png`
- `icon-192x192.png`
- `icon-384x384.png`
- `icon-512x512.png`

## Instructions

Consultez `ICONS_README.md` à la racine du projet pour les instructions détaillées de génération.

## Outils recommandés

- **En ligne** : https://realfavicongenerator.net/
- **Local** : ImageMagick, Inkscape, ou tout éditeur graphique supportant SVG
